3D Low-Poly Weapons: Midnight Deco Asset Library
Versioned 3D Assets & Kiosk UI Prototype

This repository hosts the versioned low-poly 3D assets for the Midnight Deco weapon selection module. This project demonstrates a complete end-to-end design pipeline—from modeling raw assets to implementing a functional, 3D-integrated web interface.

https://www.createdbyjerry.com/game-ui-prototype/midnight-deco

🛠 The Tech Stack
I built this project using a multi-disciplinary workflow to ensure high performance and visual fidelity:

3D Modeling: Blender (Low-poly geometry optimized for web-GL).

Texturing: Procreate (Hand-painted textures and custom iconography).

UI/UX Design: Figma (Layout and stylized visual language).

Asset Hosting: GitHub (Serving as the version-controlled CDN for .glb files).

Frontend: Webflow & Three.js (Real-time 3D rendering and interaction logic).

🌌 Concept: Midnight Deco
The prototype simulates a weapon selection terminal for a stylized kiosk gaming experience. The aesthetic blends Art Deco geometric precision with a high-contrast noir palette, creating a "diegetic" UI that feels like a physical part of the game world.

Project Goals

3D Pipeline Integration: Achieving a seamless handoff between Blender and a live browser environment.

Optimized Performance: Utilizing low-poly techniques to ensure the 3D inspection module runs at 60fps.

Functional Prototyping: Moving beyond static design to show real-time asset rotation, data feedback, and navigation.
